# SectionRecyclerViewAdapter
`针对各种复杂布局做总结，完整实现典型的场景`


> 1. 实现典型分组复杂列表适配器；<br>
> 2. 实现分组分割线；<br>
> 3. 实现上拉加载的功能，添加回调，添加滑动的容差值；<br>
> 4. 实现列数的控制；<br>
> 5. 支持设置空布局，动态监听数据<br>
> 6. 归纳复杂布局场景，使得通用性更为广泛<br>
